# 🚀 WhatsApp AI Assistant - Quick Setup (5 Minit)

## 📋 Apa Yang Anda Perlukan:
- ✅ Computer/Laptop
- ✅ Browser (Chrome/Firefox/Edge)
- ✅ WhatsApp di phone
- ✅ Internet connection

---

## ⚡ Setup dalam 5 Minit:

### Step 1: Install Tampermonkey (1 min)
1. **Buka browser** anda
2. **Pergi ke**: [https://www.tampermonkey.net/](https://www.tampermonkey.net/)
3. **Click "Install for Chrome/Firefox/Edge"**
4. **Click "Add Extension"**
5. **Restart browser** (tutup dan buka semula)

### Step 2: Copy Userscript (2 min)
1. **Klik ikon Tampermonkey** (biasanya di toolbar browser)
2. **Click "Create a new script"**
3. **Delete semua kod** yang ada
4. **Copy kod dari file `whatsapp-ai-userscript.user.js`**
5. **Paste** dalam Tampermonkey editor
6. **Click Save** (Ctrl+S atau File → Save)

### Step 3: Test di WhatsApp Web (2 min)
1. **Buka WhatsApp Web**: [https://web.whatsapp.com/](https://web.whatsapp.com/)
2. **Scan QR code** dengan phone anda
3. **Tunggu loading** (30 saat - 1 minit)
4. **Cari butang 🤖** di kanan bawah screen
5. **Click butang 🤖** untuk buka AI Assistant

---

## 🎯 Cara Test:

### Test 1: Basic Chat
1. Click butang 🤖
2. Taip: "Hai, boleh tolong buatkan saya daily schedule?"
3. Click "Hantar"
4. Tunggu response dari AI

### Test 2: WhatsApp Command
1. Dalam mana-mana WhatsApp chat
2. Taip: `@ai tolong buatkan saya todo list untuk hari ini`
3. Tunggu popup AI response muncul
4. Click "Copy" untuk copy response

---

## ✅ Success Indicators:

### ✅ Anda Berjaya Jika:
- Nampak butang 🤖 di WhatsApp Web
- Boleh click butang dan buka chat interface
- AI beri response apabila anda hantar mesej
- Popup muncul bila taip @ai dalam WhatsApp

### ❌ Masalah & Solution:

**Masalah 1: Tak nampak butang 🤖**
- Solution: Refresh WhatsApp Web page

**Masalah 2: Butang ada tapi tak berfungsi**
- Solution: Check Tampermonkey script aktif

**Masalah 3: AI tak beri response**
- Solution: Check internet connection

**Masalah 4: Popup tak muncul bila taip @ai**
- Solution: Taip @ai di chat baru, bukan reply
